# Learning Lab

A personal monorepo for experimenting with different technologies by following tutorials and building small projects.

Instead of creating multiple scattered repositories, this repo acts as a centralized learning space where each project is organized by stack and purpose.

<br />

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/chesteralejandro/learning-lab.git
```

2. Go to the specific project directory:

```bash
cd <stack-directory>/<project-name>
```

3. Read the specific project's README for further instructions.

<br />

## 🎯 Purpose

- Practice new tools and frameworks through hands-on projects
- Keep tutorial-based work separate from production-ready projects
- Maintain a clean and organized GitHub profile
- Track learning progress over time

<br />

## 📁 Folder Structure

```
📁 learning-lab/
├── README.md
└── 📁 astro/
    └── ...
```

Each project folder contains:

- Source code
- Project-specific README
- Notes or learnings (if applicable)
